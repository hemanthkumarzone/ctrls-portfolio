What is Cost Analyzer

AI cost intelligence,
not just cost reporting

Most cloud billing dashboards tell you what you spent. Cost Analyzer tells you why you spent it, what drove the change, which team owns it, and what to do about it — automatically.

From raw billing data to actionable intelligence

Cost Analyzer ingests billing exports, API usage logs, and hardware telemetry from every connected source — then normalises, attributes, and enriches the data into a unified cost model your entire organisation can act on.

Unlike generic cloud cost tools, Cost Analyzer understands the economics of AI workloads specifically — token-based billing, GPU-hour pricing, fine-tuning vs inference cost profiles, model routing decisions, and dataset storage costs.

How it works

Up and running in minutes,
insights within hours

No lengthy onboarding. No data warehousing project. No professional services required. Connect your accounts and Cost Analyzer starts working immediately.

01

Connect your sources

Link AWS, Azure, GCP, OpenAI, Anthropic, Bedrock, Vertex AI, Hugging Face, and on-prem clusters via read-only API keys or IAM roles. Takes under 10 minutes per source.

02

Ingest and normalise

Cost Analyzer pulls billing exports, usage logs, and hardware telemetry and normalises everything into a unified cost model — regardless of how each provider structures its billing.

03

Attribute and enrich

Costs are automatically attributed to teams, projects, and cost centres using existing tags, enriched with inferred ownership signals where tags are missing.

04

Analyse and alert

Dashboards, anomaly alerts, efficiency scores, and forecasts are immediately available. Set budget thresholds and receive Slack or email alerts the moment spend deviates.

05

Act and measure

Cost insights flow directly into the Optimization, Orchestration, and Waste Elimination capabilities — so savings aren't just identified, they're executed and measured.

Cost Analyzer is read-only by default. It never modifies, scales, or terminates any resource in your environment unless you explicitly connect it to our Optimization or Waste Elimination capabilities and configure automation permissions.

Cost Analyzer ingests usage logs from your AI provider accounts (OpenAI, Anthropic, Cohere, etc.) alongside the request metadata you tag at the application level. Where request-level tagging is not in place, we infer attribution from deployment names, API key ownership, and usage patterns. We also offer a lightweight SDK that adds automatic cost attribution headers to every API call without modifying your model logic.
Data freshness depends on the source. AI API usage data (token counts, call volumes) is updated every 15 minutes via direct provider API polling. Cloud billing data (AWS CUR, Azure Cost Management) is refreshed every 4–24 hours depending on how frequently your cloud provider publishes billing exports — typically every 4 hours for AWS and every 8 hours for Azure. Hardware telemetry from the on-premises agent is real-time with a 60-second resolution.
